Faculdade Saberes Universais (FSU)

	Requisitos Funcionais
		RF001 - O Sistema permitirá cadastrar o nome da disciplina com carga horária
		RF002 - O Sistema terá uma um sistema de cadastro de sala de aula permitindo informar a quantidade mínima e máxima de alunos por sala
		RF003 - O Sistema permitirá o lançamento do plano de curso descrevendo o que será ministrado nas aulas
		RF004 - O Sistema permitirá cadastrar o professor, sua competência e seus dados como telefone, e-mail, endereço, cidade e estado
		RF005 - O Sistema permitirá que cada professor possa ministrar uma ou mais disciplinas
		RF006 - O Sistema NÃO permitirá que uma disciplina tenha mais de um professor
		RF007 - O Sistema terá um controle de data inicial e final das disciplinas
		RF008 - O Sistema terá um painel para o professor atualizar suas informações de contato e endereço
		RF009 - O Sistema fará um agendamento de disciplina, permitindo que os professores se organizarem
		RF010 - O Sistema disponibilizará um cronograma para detalhar o conteúdo ministrado na aula
		RF011 - O Sistema terá um cadastro e controle de alunos contendo nome, data de nascimento e ano letivo
		RF012 - O Sistema permitirá que o professor controlar as presenças, as faltas, as notas e as observações dos alunos
		RF013 - O Sistema terá um controle de salas, 
		RF014 - O Sistema permitirá o gerenciamento de uso das salas
	Requisitos Não Funcionais
		RNF001 - O Sistema será hospedado na plataforma da empresa Umbler
		RNF002 - O Sistema será escrito na linguagem PHP e banco de dados MySQL
		RNF003 - O Sistema terá atualizações semestrais
		RNF004 - A manutenção do sistema será cobrado mensalmente com valor a ser estipulado com o cliente
		RNF005 - O Sistema terá backup automático feito pela plataforma da Umbler
		RNF006 - O Sistema só poderá ser acessado via web
		RNF007 - O Sistema será 24/7
		RNF008 - O Sistema estará vinculado as leis LGDP
