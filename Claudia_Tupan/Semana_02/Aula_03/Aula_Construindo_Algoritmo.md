        Linguagem Natural
            Ferramenta de Comunicação
                A linguagem é uma ferramenta essencial para a comunicação entre as pessoas
            Entendimento Mútuo
                O uso de uma mesma linguagem permite que as pessoas se entendam e se comuniquem de forma eficaz.
            Linguagens Naturais
                As línguas naturais são usadas de forma formal para estabelecer a comunicação entre as pessoas.
            
            NOTA: Não pode ter embiguidade. A palavra pode ter mais de um significado. Seja o mais claro possível e lembre-se, Palavra não tem tom.
