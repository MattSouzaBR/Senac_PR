CARACTERISTICAS DE Um PROGRAMA
    - Entrada: Pode ter muitas ou nenhuma entrada (nenhuma quando os dados são gerados pelo próprio algoritmo.)
    - Saída: Deve ter obrigatoriamente uma saída
    - Linguagem de Programação: é um método padronizado para comunicar instruções para o computador
    - Finitude: deve ter um número finito de passos para obter a resposta
    - Generalidade: Deve servir apra uma classe de problemas
    - Clareza: Cada passo deve ser definido de forma clara e precisa sem ambiguidade.
