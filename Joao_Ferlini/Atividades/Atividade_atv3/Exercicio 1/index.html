Sobre o Curso

O Curso Técnico em Informática prepara os alunos para atuar no desenvolvimento de sistemas, manutenção de computadores e criação de páginas web.

Principais Disciplinas

  • HTML e CSS

    Aprenda a desenvolver páginas web utilizando estrutura HTML e estilização com CSS.

  • Lógica de Programação

    Desenvolva o pensamento lógico para resolver problemas utilizando algoritmos.

  • Banco de Dados

    Aprenda a modelar e manipular dados utilizando sistemas gerenciadores de banco de dados.
